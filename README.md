# 📚 Prueba Parcial — Simulacro POO en Java

Repositorio de simulacro para la evaluación parcial del curso de **Programación Orientada a Objetos (POO)** con Java. Contiene un proyecto Java estructurado bajo los principios fundamentales de la POO.

---

```

├── ProyectoPOO/        # Proyecto principal en Java
│   └── src/            # Código fuente
├── .gitignore
├── LICENSE
└── README.md
```

## 🛠️ Tecnologías utilizadas

- **Lenguaje:** Java
- **Paradigma:** Programación Orientada a Objetos (POO)
- **IDE recomendado:** IntelliJ IDEA / Eclipse / NetBeans

## 🚀 Cómo ejecutar el proyecto

### Prerrequisitos

- Tener instalado el [JDK 8 o superior](https://www.oracle.com/java/technologies/downloads/)
- Un IDE compatible con Java o acceso a la terminal

### Pasos

1. **Clonar el repositorio:**

   ```bash
   git clone https://github.com/Antomaker/prueba_parcial.git
   cd prueba_parcial
   ```

2. **Abrir el proyecto en tu IDE:**
   - Abre la carpeta `ProyectoPOO` como proyecto Java en tu IDE favorito.

3. **Compilar y ejecutar:**
   - Desde el IDE, ejecuta la clase principal (`main`).
   - O desde la terminal:
     ```bash
     cd ProyectoPOO/src
     javac *.java
     java Main
     ```

## 📖 Conceptos de POO aplicados

Este simulacro pone en práctica los siguientes conceptos:

- **Encapsulamiento** — Uso de modificadores de acceso y métodos getter/setter.
- **Herencia** — Jerarquías de clases con `extends`.
- **Polimorfismo** — Sobrescritura y sobrecarga de métodos.
- **Abstracción** — Clases abstractas e interfaces.
